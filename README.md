# Desafio Zapt

Desafio técnico do processo seletivo da Zapt.

## Como rodar o projeto?

1. Faça o clone do repositório:
```bash
git clone https://github.com/huferr/desafio-zapt.git
```
2. Navegue até o diretório:
```bash
cd desafio-zapt
```
4. Instale as dependências:
```bash
npm install
```
4. Rode o app:
```bash
npm start
```
